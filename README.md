1. Развёртывание проекта на сервере.
2. Переходим в корневую папку проекта
3. Запускаем следующую команду:
    docker compose up -d 
4. Ожидаем скачивание образов и установки зависимостей
5. Необходимо создать базу данных:
    - можно импортировать бэкап из папки backup, используя следующую комманду:
        psql -U valeron -d tree -h 172.40.1.3 -f ./backup/tree.sql
    - можно запустить миграции использую клманду:
        alembic upgrade head
6. Если использовать alembic, то будет создана пустая база данных, 
данные нужно будет добавлять ручками. Для добавления данных можно использовать 
скрипт data.py
7. Если же использовать файл tree.sql, то в файле уже будут тестовые данные.
8. Необходимо активировать вирутальное окружение:
    source .venv/bin/activate

9. Запустить сервис, выполнив python main.py
